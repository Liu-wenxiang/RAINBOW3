一个用于控制 RAINBOW3 设备灯效的 Windows 图形界面工具，支持模式灯颜色、14 颗灯珠颜色、多色效果、配置保存、程序联动自动切换等功能。

## 项目说明

本项目当前版本的功能实现、界面迭代、打包流程与文档整理由 GitHub Copilot 协助完成。

## 功能特性

- 支持三种模式灯颜色自定义
- 支持 14 颗灯珠单独设置颜色
- 支持常亮、呼吸、交替、渐变模式
- 支持每颗灯珠 1 到 5 种颜色的多色配置
- 支持多色格式刷，快速复制灯珠多色方案
- 支持左右镜像联动
- 支持保存多套配置方案
- 支持关联程序自动切换配置
- 支持打包为 Windows EXE 独立运行

## 目录结构

- [hid_light_gui.py](hid_light_gui.py)：主图形界面程序
- [send_hid.py](send_hid.py)：底层 HID 发包脚本
- [RAINBOW3_GUI.spec](RAINBOW3_GUI.spec)：PyInstaller 打包配置

## 基本使用方法

### 1. 模式灯颜色

- 在右侧模式灯区域设置 Switch、PS、Xbox 三个模式灯颜色
- 点击圆点可直接选色
- 也可以直接输入 6 位十六进制颜色值
- 点击“应用”后发送到设备

### 2. 14 灯珠颜色

- 点击主界面灯珠可选中灯珠并填充当前颜色
- 点击空白处可取消选择
- 点击“全部选择”可一次选中全部灯珠
- 开启“左右同色”后，左右对应灯珠会同步设置

### 3. 多色编辑

- 选中一个或多个灯珠后，点击“编辑多色”
- 在弹出的窗口中填写 1 到 5 个颜色
- 保存后会写入对应灯珠的多色方案

### 4. 多色格式刷

- 先选中一颗已配置好多色方案的灯珠
- 点击“多色格式刷”
- 再点击目标灯珠即可复制该灯珠的多色配置
- 按 Esc 或再次点击按钮可退出格式刷模式

### 5. 应用灯效

- 选择常亮、呼吸、交替或渐变模式
- 调整亮度与动效速度
- 点击左侧区域中的“应用”发送当前 14 灯珠效果

## 配置方案使用方法

### 保存配置

- 在右侧“配置方案”区域输入配置名称
- 点击“保存”

### 加载配置

- 从下拉框中选择配置
- 点击“加载”

### 删除配置

- 选择对应配置后点击“删除”

### 启动默认加载

- 程序启动时会默认加载第一个已保存配置

## 程序联动自动切换

每个配置都可以绑定一个程序名。

使用方法：

- 在“关联程序”中填写 EXE 名称，例如 game.exe
- 保存当前配置
- 当检测到该程序正在运行时，软件会自动加载并应用对应配置

说明：

- 自动检测为后台轮询执行
- 已做无窗口处理，不会周期性弹出命令窗口
- 如果多个配置绑定了程序，按当前配置遍历顺序匹配第一个命中的配置

## 颜色格式说明

- 使用 6 位 RGB 十六进制颜色，例如 ff0000、00ffcc、ffffff
- 可输入带或不带 # 的颜色值，程序内部会自动规范化

## 常见问题

### 1. 无法发送 HID 数据

请确认：

- 设备已连接
- 当前系统已识别对应 HID 设备
- Python 环境中已安装 hidapi

### 2. 打包后配置无法保存

当前版本已将配置文件路径适配为：

- 源码运行时保存在脚本目录
- EXE 运行时保存在 EXE 所在目录

### 3. EXE 启动较慢

单文件 EXE 首次启动或每次启动时会有解包过程，这是 PyInstaller onefile 模式的正常现象。
